Goal

The Skateboard Builder application will allow users to build their own custom skateboard from a vareity of different components. The app will allow the user to save multiple skateboard designs. It will also allow them to purchase the parts so that they can build it in real life. The app is for a fake business that sells skateboard components.

The primary feature of the application is an interactive 3D view that presents the skateboard designs or individual components. The intention is that anyone can use the app (not just customers) because it will increase the popularity and amount of feedback for it.

The Skateboard builder app should provide the user the following functionalities:

  • Save Skateboard Designs
  • Open SkatebBoards Designs
  • Design a skateboard
    • Select Deck Design
    • Select Trucks
    • Select Wheels
    • Select Griptape
    • Select Bearings
  • View the design in 3D
  • View individual components in 3D
  • View Part Summary
  • Order parts
